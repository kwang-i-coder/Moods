## 1️⃣ Overview
### *공부 무드, 공간으로 완성되다*

*공부 분위기 나는 카페는 없나?*
**오늘 카공은 어디서 하지?**
**시간만 말고 공부한 이 느낌을 기억하고 싶어!**

**무드**는 나에게 맞는 공부 공간을 찾고 싶어하는 사람들을 대상으로 만든 서비스입니다.
**공부 장소는**는 **공부 능률**에 영향을 미친다.
시각적으로 보이고 쉽게 파악할 수 있는 것이 **시간**이기 때문이다.
공부에 필요한 것은 하나만이 아닙니다.
공부 집중을 돕는 **다양한 요소를 고려**하여,
**나에게 맞는 공부 공간**을 찾아가 보세요.
